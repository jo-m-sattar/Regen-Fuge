Future Recommendations

Although the project has reached its conclusion, several improvements can be implemented to enhance performance, safety, and usability. The following recommendations are proposed for future development.

Auto-Balancing System

Implement an auto-balancing system capable of detecting rotor imbalance and excessive vibration. This system would automatically reduce speed or stop operation before test tubes break or burst. An accelerometer sensor such as the MPU6050 can be used to detect vibration levels.

Dynamic Motor Cooling

Add a dynamic cooling system to monitor motor temperature and activate cooling fans when overheating is detected. This can be achieved using an NTC thermistor to ensure safe operating temperatures and extend motor lifespan.

BLDC Motor Upgrade

Replace the universal motor with a Brushless DC (BLDC) motor, which offers higher efficiency, lower heat generation, and more stable rotation due to its electronically controlled, frictionless design.

Monitoring & Control System

Integrate a monitoring system using an Arduino or ESP32 microcontroller to display and transmit real-time operational data such as rotational speed, temperature, and operating duration through a simple application.

Safety Lid Mechanism

Add a transparent safety lid equipped with a safety interlock switch that automatically stops the motor if the lid is not fully closed, ensuring operator safety during high-speed operation.
